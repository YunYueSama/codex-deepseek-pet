## 正文

最近做了一只会在桌面陪你工作的 **DeepSeek 拟人化宠物**。

她是一只蓝发、鲸鱼尾巴、穿深蓝白色女仆装的“蓝色大肥鱼”。平时会待在桌面角落看着鼠标，偶尔散步、挥手、思考或者偷偷睡觉。项目叫 **codex-deepseek-pet**，目前已经开源，也提供了 Windows 安装包。

> 我不是吃白饭的大肥鱼！

![项目运行效果](./preview.png)

### 最想做好的功能：不是眼球平移，而是真的转头

早期版本只是在正面立绘上移动瞳孔，方向变化很不明显，还容易在眼睛旁边出现两个深蓝色圆点。后来我把这套方案整个删掉，重新制作了 **16 张完整方向立绘**。

现在鼠标绕着宠物移动时，会以正上方为 `0°`，每隔 `22.5°` 切换一张图。变化的不只是眼睛，头部朝向、面部透视、眼睑和身体姿态都会一起变化，也没有使用 CSS 旋转或拉伸去硬掰正面图。

### 除了看鼠标，她还会做什么

- 单击会随机回应，双击会开心蹦起来。
- 可以按住拖到桌面任意位置，拖动时还有专属反应。
- 会自动在当前显示器上散步，并根据方向切换左右跑步动作。
- 有待机、思考、开心、害羞、兴奋、挥手、惊讶、审阅、跳跃、睡眠等动作。
- 托盘菜单可以控制置顶、鼠标穿透、自动散步、开机启动、尺寸和位置。
- 支持多显示器，重启后会记住设置。

### 安装和隐私

普通用户直接到 GitHub Release 下载：

**项目地址：** https://github.com/YunYueSama/codex-deepseek-pet  
**最新安装包：** https://github.com/YunYueSama/codex-deepseek-pet/releases/latest

下载 `codex-deepseek-pet-Setup-X.X.X-x64.exe` 后正常安装即可，不需要 Node.js，也不需要 `OPENAI_API_KEY`。

项目完全在本地运行，没有联网请求和遥测。系统鼠标坐标只用于本机动画，不会保存或上传。安装包暂时没有商业代码签名证书，Windows SmartScreen 首次运行时可能会弹出保护提示，介意的话也可以直接查看源码后自行构建。

另外，仓库里还提供了可导入 Codex / ChatGPT 桌面版的 v2 自定义宠物包，包含 9 行标准动作和 16 个注视方向。独立 Electron 桌宠与 Codex 内置宠物是两种运行方式，可以按自己的使用场景选择。

这是个人制作的非官方同人项目，与 DeepSeek 官方无隶属关系。欢迎体验，也欢迎在评论区告诉我还想让“大肥鱼”增加什么动作。
