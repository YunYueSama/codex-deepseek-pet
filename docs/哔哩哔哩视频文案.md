# 哔哩哔哩视频文案与分镜

## 视频定位

- 建议时长：约 3 分钟。
- 视频节奏：前 10 秒直接展示效果，随后解释真实 16 方向、互动动作、安装方式和本地隐私。
- 画面比例：`1920x1080`，录屏时隐藏个人文件、通知、账号信息和浏览器收藏栏。
- 核心卖点：DeepSeek 拟人化宠物、完整身体转向、丰富动作、完全本地运行、开源且有 Windows 安装包。

## 视频简介

做了一只会在桌面陪你工作的 **DeepSeek 拟人化宠物**。

她会读取系统级鼠标位置，用 16 张完整立绘让头部、面部和身体一起转向；还支持单击回应、双击跳跃、拖拽、自动散步、随机动作、托盘设置和多显示器。项目完全在本地运行，不需要 `OPENAI_API_KEY`，也不会上传鼠标坐标。

项目源码：  
https://github.com/YunYueSama/codex-deepseek-pet

Windows 安装包：  
https://github.com/YunYueSama/codex-deepseek-pet/releases/latest

这是个人制作的非官方同人项目，与 DeepSeek 官方无隶属关系。安装包暂未使用商业代码签名证书，首次运行可能出现 Windows SmartScreen 提示，请确认从项目 Release 页面下载。

## 录制素材清单

1. 一段 16 方向系统鼠标环绕录屏，建议录制两遍，选择切帧最稳定的一遍。
2. 一段鼠标位于其他应用窗口时仍持续注视的录屏。
3. 单击、双击、拖拽三种交互各一段。
4. 自动散步和右键托盘设置各一段。
5. 思考、开心、害羞、挥手、惊讶、审阅、睡眠、左右跑步动作各一段。
6. GitHub Release 下载与 Windows 安装过程录屏。
7. Codex / ChatGPT 自定义宠物设置页面录屏。
8. 静态图：`docs/preview.png`、`docs/gaze-16-directions.png`、核心灵感图和三张动作图。

## 后期注意事项

- 不要用转场效果掩盖 16 方向切换，核心演示尽量采用一镜到底的真实鼠标移动。
- 方向总览至少停留 5 秒，确保观众能看清头部和身体变化。
- 所有软件窗口、浏览器账号、邮箱、文件路径和通知都要提前脱敏。
- 项目地址同时放在视频简介、置顶评论和片尾，避免只出现在口播里。
- 明确写“非官方同人项目”和“安装包暂未商业签名”，避免观众误解。
